# homebrew-tsm

Homebrew tap for [`tsm`](https://github.com/adibhanna/tsm).

## Install

```bash
brew tap adibhanna/homebrew-tsm
brew install tsm
```

## Notes

- The Homebrew formula installs the portable fallback backend.
- For the full Ghostty-backed build, use a downloadable release archive from the main repository.
